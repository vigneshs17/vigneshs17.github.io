Optical Number Recognition
This project aims at an implementation of a system that will take a input image with hand written number on a fragment of paper. The system will be designed to recognize these numbers and store them. The input would be from a handheld device like a mobile camera. The project deals with taking the input , normalizing aspect ratio and scaling, feature extraction and recognizing the number. We would be using kNN to implement it and the MNIST database. We achieved an accuracy of 80%.
