Projects

TravelPool

The taxi fare in and around Mangalore is insane. Students have to spend a lot of money to go to the airport. This web application seeks to solve that problem by enabling students to form travel pools so that they can save money by sharing a ride with other students with similar travel needs. This was built using Ruby-on-Rails.

Optical Number Recognition

This project implements a system that will take an input image with a hand-written number. It outputs a digital form of the hand-written number. We implemented it using kNN and the MNIST database. We achieved an accuracy of 80%.

Todo List App

This todo list app enables a user to add, edit, delete or complete a todo item. This runs on a local host and was implemented using Node.js and Vue. The database used was MongoDB.

Blog App

This blog app allows you to create a blog post. You can also edit it or delete it. This was implemented using Node.js and ReactJS. I also included authentication; a user has to login in order to post a blog.
